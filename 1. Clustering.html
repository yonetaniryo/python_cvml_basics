クラスタリング¶

ある決まった特徴の組で表現されるサンプル（たとえば身長と体重で特徴づけられる個人）が複数与えられているとします．このようなサンプルの集合を，いくつかのグループに分ける問題をクラスタリングと言います．以下では最も有名なクラスタリング手法の一つであるk-meansクラスタリングを使ってみましょう．

In [1]:
from sklearn.datasets import make_blobs  # クラスタリングのための人工データを生成する関数
import matplotlib.pyplot as plt
%matplotlib inline

x, _ = make_blobs(cluster_std=.5)
plt.scatter(x[:, 0], x[:, 1], 10)
Out[1]:
<matplotlib.collections.PathCollection at 0x1a123db3c8>

上のサンプル分布をみると，どうもこのサンプル集合は3つのグループに分かれそうです．

k-meansクラスタリングは，以下の手続きによってサンプル集合をグループ（クラスタ）に分割します．

  1. （初期化）適当なクラスタ中心（centroid）をkを決める
  2. （割当）各サンプルを最も中心が近いクラスタに割り当てる
  3. （クラスタ更新）各クラスタについて，中心を割り当てられたサンプルの平均に置き換える．
  4. 手続き1. 2.を繰り返す
In [2]:
from sklearn.cluster import MiniBatchKMeans

KM = MiniBatchKMeans(n_clusters=3)
idx = KM.fit_predict(x)
plt.scatter(x[:, 0], x[:, 1], 10, idx)
Out[2]:
<matplotlib.collections.PathCollection at 0x1a1641c748>

確かに3つのグループに分類することができました．

課題¶

  • make_blobsのn_samples, cluster_stdを変化させ，分布がどのように変わるかを見てみましょう．また，これらの変化に対してk-meansの出力結果，計算速度がどのように変化するか見てみましょう．
  • k-meansは事前にクラスタ数を決定する必要があります．KMeansのn_clustersを2や4に変化させたとき，出力結果，計算速度がどのように変化するか見てみましょう．

クラスタリングの応用: 画像細分割（superpixel segmentation）¶

クラスタリングを応用した画像処理技術の一つにSLICと呼ばれる画像細分割法があります．画像細分割とは，与えられた画像をパズルピースのような小領域（superpixel）に分割する技術で，この後に述べる重要領域抽出などで役立ちます．

SLICにおいて小領域は，互いに位置が近く，色が似ているピクセルの集合で構成されます．ということは，画像中の各ピクセルを位置，色を属性としたサンプルとみなし，上のクラスタリング手法を適用することで，小領域が得られそうです．

クラスタリングの応用: 重要領域検出（salient object detection）¶

クラスタリングを応用したより実践的なコンピュータビジョン技術の一つに，重要領域検出があります．これは，与えられた画像において重要な領域を自動的に抽出する技術です．ここでは，入力画像について「希少な色を持つ小領域ほど重要」という仮定のもと，重要領域を抽出してみましょう．

In [8]:
sp_ = sp.flatten()
color = rgb2lab(img).reshape((img.shape[0] * img.shape[1], 3))  # Lab表色系での色情報
color_sp = np.vstack([color[sp_ == x].mean(axis=0) 
                      for x in np.unique(sp_)])  # 各小領域における平均色

各小領域の持つ色の希少さを判断するためには，画像全体にわたってどのような色が現れやすいかという確率分布（たとえば赤は70%，緑は30%，…）を推定する必要があります．ここでは，scikit-learnに実装されているカーネル密度推定sklearn.neighbors.KernelDensityと呼ばれる手法を利用してみましょう．

In [9]:
from sklearn.neighbors import KernelDensity

kd = KernelDensity(bandwidth=1e1)
kd.fit(color_sp)  # 確率密度分布の推定
color_prob = np.exp(kd.score_samples(color_sp))  # 各色が現れる確率の計算．kd.score_samplesは確率の対数を出力する
importance = 1. / color_prob  # 希少＝確率が低いほど重要度が高くなる
so = importance[sp_].reshape((img.shape[0], img.shape[1]))  # 画像の形に整形

plt.figure(figsize=[12, 8])
plt.subplot(1, 3, 1)
plt.imshow(img)
plt.subplot(1, 3, 2)
plt.imshow(so)
plt.subplot(1, 3, 3)
img_ = img.copy()
img_[so < np.mean(so)] = 0
plt.imshow(img_)
Out[9]:

課題¶

  • 様々な画像に対して上記のプログラムを実行し，どのような画像において直感とあった領域が検出される・されないか考察してみましょう．
  • 上の結果では，背景の一部が重要領域として検出されてしまっています．猫の領域のみをより高精度に検出するにはどういったアプローチが考えられるでしょうか．

最終課題¶

  1. 入力画像に対して重要領域を抽出する関数を作成しましょう．その際，superpixelの数やカーネル密度推定のbandwidthをパラメタとして設定できるようにしましょう．
  2. 1.を拡張し，映像を入力すると各フレームについて重要領域を抽出する関数を作成しましょう．また，この処理を複数フレームで並列して実行できるように拡張しましょう．
  3. make_blobsで与えられたサンプル集合について，最も適切なクラスタ数を推定する手法を考え，実装しましょう
  4. （発展課題）2., 3.を利用し，入力映像に含まれる重要な物体の数をカウントし，各物体について代表的な画像を表示する手法を考え，実装しましょう．たとえば，複数フレームにわたって同一の猫が現れる場合，それらは一種類の物体としてカウントするものとします．
